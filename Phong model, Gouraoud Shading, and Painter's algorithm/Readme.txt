Type 1 to choose "cube_and_icosahedron.txt" or type 2 to choose "bunny.txt".

Then, choose whether to apple Phong Model. Type y or Y, n or N.

If no, you will see the shape(s) that have already applied the painting algorithm.
If yes, you are in Phong model setting.

Phong Model: 
	First, type kaIa(one constant between 0 to 1), Il, K.
	Then, type f(fx, fy, fz), ks(ksRed, ksGreen, ksBlue), l(lx, ly, lz)

	After that, you will see updated shapes.

Type 'E' on the output screen instead of command line to exit the program.
