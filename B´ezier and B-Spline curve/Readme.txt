First, type the display resolution. Recommend 1000.

Then, for the first curve, choose one algorithm to fulfill the curve. '1' for Bezier, '2' for B-Spline.

	If you choose '2', you need type k value.

	Then, click any point on the screen to add a point. 

	If you want to delete or move a point, type 't' on screen first. Then on the command line, type the index of this point. 
	
		To delete this point, type 'd' in command line to delete this point.
	
		To move this point, type 'm'. Then, type the coordinates of the new point.

If you want to add curve, type 'n' on the screen. Then, when you add new points, it will begin a new curve.
